---
title: "Workspaces"
description: "List and retrieve workspaces the authenticated user belongs to."
---

A workspace represents an organization in Vector. Users can belong to multiple workspaces via `WorkspaceMember`, each with a role (Admin or Member).

Workspace endpoints are **not** workspace-scoped — they live at the top level.

## List workspaces

```
GET /v1/workspaces
```

Returns all workspaces the authenticated user has access to.

### Response

```json
{
  "data": [
    {
      "id": "ws_abc123",
      "name": "Mews",
      "slug": "mews",
      "logo": "https://cdn.vectorapp.io/logos/mews.png",
      "role": "Admin",
      "createdAt": "2025-01-15T10:00:00Z",
      "updatedAt": "2025-01-15T10:00:00Z"
    }
  ],
  "hasMore": false,
  "nextCursor": null
}
```

The `role` field reflects the authenticated user's role in that workspace.

## Get workspace

```
GET /v1/workspaces/{slug}
```

### Path parameters

| Parameter | Type | Description |
|---|---|---|
| `slug` | string | Workspace slug |

### Response

```json
{
  "id": "ws_abc123",
  "name": "Mews",
  "slug": "mews",
  "logo": "https://cdn.vectorapp.io/logos/mews.png",
  "members": [
    { "personId": "per_abc123", "name": "Matt Welle", "initials": "MW", "role": "Admin" },
    { "personId": "per_def456", "name": "Jakub Novák", "initials": "JN", "role": "Member" }
  ],
  "createdAt": "2025-01-15T10:00:00Z",
  "updatedAt": "2025-01-15T10:00:00Z"
}
```

The full member list is only returned on the single-workspace endpoint, not on the list.

## Manage workspace members

### Add member

```
POST /v1/workspaces/{slug}/members
```

Requires `Admin` role in the workspace.

| Field | Type | Required | Description |
|---|---|---|---|
| `personId` | string | yes | Person to add |
| `role` | string | yes | `Admin` or `Member` |

### Update member role

```
PATCH /v1/workspaces/{slug}/members/{personId}
```

Requires `Admin` role.

| Field | Type | Required | Description |
|---|---|---|---|
| `role` | string | yes | `Admin` or `Member` |

### Remove member

```
DELETE /v1/workspaces/{slug}/members/{personId}
```

Requires `Admin` role. Cannot remove the last admin.

## Fields

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique identifier |
| `name` | string | Workspace name |
| `slug` | string | URL-friendly identifier (unique) |
| `logo` | string \| null | URL to the workspace logo |
| `members` | WorkspaceMember[] | Members with roles (on get only) |
| `createdAt` | string | ISO 8601 timestamp |
| `updatedAt` | string | ISO 8601 timestamp |

A **WorkspaceMember** object:

| Field | Type | Description |
|---|---|---|
| `personId` | string | Person ID |
| `name` | string | Person's name |
| `initials` | string | Person's initials |
| `role` | string | `Admin` or `Member` |
