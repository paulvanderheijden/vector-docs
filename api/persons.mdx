title: "Persons"
description: "Manage people in a workspace."
---

A person represents a user in Vector. People can belong to multiple workspaces. Person endpoints are workspace-scoped â€” they return persons who are members of the given workspace.

## List persons

```
GET /v1/workspaces/{workspaceSlug}/persons
```

Returns all persons who are members of the workspace.

### Query parameters

| Parameter | Type | Description |
|---|---|---|
| `role` | string | Filter by workspace role: `Admin`, `Member` |
| `cursor` | string | Pagination cursor |
| `limit` | integer | Page size (default 50, max 200) |

### Response

```json
{
  "data": [
    {
      "id": "per_abc123",
      "name": "Matt Welle",
      "email": "matt@mews.com",
      "initials": "MW",
      "role": "Admin",
      "createdAt": "2025-01-15T10:00:00Z",
      "updatedAt": "2025-01-15T10:00:00Z"
    }
  ],
  "hasMore": false,
  "nextCursor": null
}
```

The `role` field reflects the person's role in the current workspace.

## Get person

```

```

### Path parameters

| Parameter | Type | Description |
|---|---|---|

### Response

```json
  "role": "Admin",
  "createdAt": "2025-01-15T10:00:00Z",

```

## Create person

```
POST /v1/workspaces/{workspaceSlug}/persons
```

Creates a new person and adds them as a member of the workspace. Requires `Admin` role.

### Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | string | yes | Full name |
| `email` | string | yes | Email address (unique across all workspaces) |
| `initials` | string | yes | Display initials (e.g., "MW") |
| `role` | string | no | Workspace role: `Admin` or `Member` (default `Member`) |

### Response

Returns the created person with `201` status.

## Update person

```
PATCH /v1/workspaces/{workspaceSlug}/persons/{id}
```

### Body

| Field | Type | Description |
|---|---|---|
| `name` | string | Full name |
| `email` | string | Email address |
| `initials` | string | Display initials |

### Response

Returns the updated person.

## Fields

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique identifier |
| `name` | string | Full name |
| `email` | string | Email address |
| `initials` | string | Display initials |
| `role` | string | Role in the current workspace (`Admin` or `Member`) |
| `createdAt` | string | ISO 8601 timestamp |
| `updatedAt` | string | ISO 8601 timestamp |
