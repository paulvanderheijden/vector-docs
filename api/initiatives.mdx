## List initiatives

```
GET /v1/workspaces/{workspaceSlug}/initiatives
```

### Query parameters

| Parameter | Type | Description |
|---|---|---|
| `status` | string | Filter by status: `Planned`, `Active`, `Completed` |
| `visibility` | string | Filter by visibility: `Public`, `Restricted` |
| `cursor` | string | Pagination cursor |
| `limit` | integer | Page size (default 50, max 200) |

### Response

```json
{
  "data": [
    {
      "id": "ini_abc123",
      "name": "Launch DACH market by end of Q1",
      "status": "Active",
      "targetDate": "2025-03-31",
      "visibility": "Public",
      "members": [
        { "id": "per_abc123", "name": "Matt Welle", "initials": "MW" }
      ],
      "createdAt": "2025-01-15T10:00:00Z",
      "updatedAt": "2025-02-01T14:00:00Z"
    }
  ],
  "hasMore": false,
  "nextCursor": null
}
```

## Get initiative

```
GET /v1/workspaces/{workspaceSlug}/initiatives/{id}
```

### Path parameters

| Parameter | Type | Description |
|---|---|---|
| `workspaceSlug` | string | Workspace slug |
| `id` | string | Initiative ID |

### Response

Returns the full initiative object, including nested `projects`.

## Create initiative

```
POST /v1/workspaces/{workspaceSlug}/initiatives
```

### Body

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | string | yes | Initiative name |
| `status` | string | yes | `Planned`, `Active`, or `Completed` |
| `targetDate` | string | yes | Target date (flexible: month, quarter, or specific date) |
| `visibility` | string | yes | `Public` or `Restricted` |
| `memberIds` | string[] | no | Person IDs to add as members |

### Response

Returns the created initiative with `201` status.

## Update initiative

```
PATCH /v1/workspaces/{workspaceSlug}/initiatives/{id}
```

### Body

| Field | Type | Description |
|---|---|---|
| `name` | string | Initiative name |
| `status` | string | `Planned`, `Active`, or `Completed` |
| `targetDate` | string | Target date |
| `visibility` | string | `Public` or `Restricted` |
| `memberIds` | string[] | Person IDs (full replacement) |

### Response

Returns the updated initiative.

## Fields

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique identifier |
| `name` | string | Initiative name |
| `status` | string | `Planned`, `Active`, or `Completed` |
| `targetDate` | string | Target date |
| `visibility` | string | `Public` or `Restricted` |
| `members` | Person[] | Initiative members |
| `projects` | Project[] | Nested projects (on get) |
| `createdAt` | string | ISO 8601 timestamp |
| `updatedAt` | string | ISO 8601 timestamp |
