## App Shell

Wraps all pages. Always rendered.

```
App
├── SidebarProvider
│   ├── AppSidebar
│   │   ├── Sidebar > SidebarHeader (workspace logo + name)
│   │   └── SidebarContent
│   │       ├── SidebarGroup: top links (Inbox, My Items)
│   │       ├── Collapsible: Workspace (Strategy, Initiatives, Projects, Circles)
│   │       └── Collapsible: Your Circles
│   │           └── Collapsible (per circle)
│   │               ├── Sessions > Previous / Current (if exists) / Upcoming
│   │               ├── Decisions
│   │               └── Commitments
│   └── SidebarInset
│       ├── [sticky] PageHeader (injected by current page via usePageHeader)
│       └── ScrollArea
│           └── <Outlet /> → current page
```

```

```

## Decision Detail `/circles/:slug/decisions/:decisionSlug`

Single-page decision view with always-open right sidebar.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    └── PageHeaderGroup
        ├── Circle icon
        └── Breadcrumb: "Circle > Decisions > DEC-N" + statement

Page body (max-w-800, mr-72 for sidebar)
├── TextField (editable, decision statement)
├── "Rationale" → MarkdownEditor
├── "Discussion" → MarkdownEditor
├── "Summary" → read-only statement text
├── Checkbox: "Notify [requestor name]"
├── "Commitments" → CommitmentsTable (showCircle=false, noDividers)
└── "Activity" → ChangeLog

Sidebar (fixed right, 288px, always open)
└── SidebarSection "Properties"
    └── PropertiesRow (stacked)
        ├── PropertiesRow.Item "Status" → StatusField (editable)
        ├── PropertiesRow.Item "Requested" → PersonField (read-only, showName, title="Requested by")
        ├── PropertiesRow.Item "Age" → colored text
        ├── PropertiesRow.Item "Circle" → Link
        ├── PropertiesRow.Item "Sponsor" → PersonField (nullable, editable, showName, title="Sponsor")
        └── PropertiesRow.Item "Owner" → PersonField (nullable, editable, showName, title="Owner")
```

## Commitment Review `/circles/:slug/commitments/:commitmentSlug`

Commitment view with tabs (overview, activity) and right sidebar.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    ├── PageHeaderGroup
    │   ├── Circle icon
    │   ├── Breadcrumb: "Circle > Commitments > COM-N" + outcome
    │   └── Tab buttons: Overview | Activity
    └── PageHeaderGroup
        ├── [Re-commit?] button (if rolled/at-risk)
        └── Sidebar toggle

Page body (conditional on active tab)
├── Overview tab (max-w-800)
│   ├── Alert banner (rolled = red, at-risk = amber)
│   ├── TextField (editable, commitment outcome)
│   ├── PropertiesRow (title="Properties") → StatusField (commitment), due date, owner, circle
│   └── "Linked Decision" → Link card with StatusField (decision)
└── Activity tab (max-w-800)
    └── ChangeLog

DetailSidebar (fixed right, 288px, toggle)
├── SidebarSection "Properties"
│   └── PropertiesRow (stacked)
│       ├── PropertiesRow.Item "Status" → StatusField (commitment)
│       ├── PropertiesRow.Item "Due" → text
│       ├── PropertiesRow.Item "Owner" → text
│       └── PropertiesRow.Item "Circle" → Link
└── SidebarSection "Linked Decision" → Link + StatusField (decision)
```

## Session Detail `/circles/:slug/sessions/:sessionNumber`

Meeting workspace with decisions agenda, commitments review, notes editor, and decision drawer.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    └── PageHeaderGroup: circle icon + "Circle > Session N" + friendly date

Page body (max-w-800)
├── TextField (editable session title, as h1)
├── PropertiesRow (title="Properties")
│   ├── PropertiesRow.Item → DateField (session date)
│   ├── PropertiesRow.Item → PropertySelect (Open / Closed)
│   └── PropertiesRow.Item → PersonField (chair, editable, showName)
├── "Decisions" section
│   ├── Section header + [Add Decision] button
│   └── DecisionsTable
│       (showCircle=false, showAge=false, showSelect=false,
│        showDragHandle=false, showHeader=false, combinedStatusOwner,
│        noDividers, onReorder, onDecisionClick → opens drawer)
├── "Commitments" section
│   ├── Section header + [Add Commitment] button
│   └── CommitmentsTable
│       (showCircle=false, showDecision=false, showSelect=false,
│        showHeader=false, combinedStatusOwner, noDividers)
└── "Meeting Notes" section
    └── MarkdownEditor (session notes)

Decision drawer (fixed right, 480px, toggle on row click)
├── Drawer header: ref + statement + [open full] Link + [close] button
└── Drawer content
    ├── Properties row: StatusField (decision), age, owner, requested by
    ├── "Rationale" → text
    ├── "Implications" → bullet list
    └── "Discussion" → text
```

## Project Detail `/projects/:projectId`

Project detail with tabs (overview, decisions, commitments) and right sidebar.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    ├── PageHeaderGroup
    │   ├── Breadcrumb: "Projects >" + title
    │   └── Tab buttons: Overview | Decisions | Commitments
    └── PageHeaderGroup
        ├── [Follow/Following] button (public projects, non-owner)
        └── Sidebar toggle

Page body (conditional on active tab)
├── Overview tab (max-w-800)
│   ├── Alert banner (if >2 rolled commitments)
│   ├── TextField (editable, project title)
│   ├── TextField (description, as p)
│   ├── PropertiesRow (title="Properties")
│   │   ├── PropertiesRow.Item → PropertySelect (health)
│   │   ├── PropertiesRow.Item → PropertySelect (visibility)
│   │   ├── PropertiesRow.Item → PropertySelect (owner)
│   │   └── PropertiesRow.Item → Link to parent initiative
│   └── "Teams" → TeamChips
├── Decisions tab
│   └── DecisionList
└── Commitments tab
    └── CommitmentList

DetailSidebar (fixed right, 288px, toggle)
├── SidebarSection "Properties"
│   └── PropertiesRow (stacked)
│       ├── PropertiesRow.Item "Health" → HealthIndicator
│       ├── PropertiesRow.Item "Visibility" → PropertySelect
│       ├── PropertiesRow.Item "Owner" → PropertySelect
│       └── PropertiesRow.Item "Initiative" → Link
├── SidebarSection "Signals"
│   └── PropertiesRow (stacked)
│       ├── PropertiesRow.Item "Open" → commitment count
│       ├── PropertiesRow.Item "Rolled" → rolled count
│       └── PropertiesRow.Item "Dec. (14d)" → recent decision count
├── SidebarSection "Teams" → TeamChips
└── SidebarSection "People" (private projects only)
    ├── Owner row: Avatar + name + "Owner" Badge
    ├── Subscriber rows: Avatar + name + [remove] button
    └── Select (invite person)
```
