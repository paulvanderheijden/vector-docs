## App Shell

Wraps all pages. Always rendered.

```
App
├── SidebarProvider
│   ├── AppSidebar
│   │   ├── Sidebar > SidebarHeader (workspace logo + name)
│   │   └── SidebarContent
│   │       ├── SidebarGroup: top links (Inbox, My Items)
│   │       ├── Collapsible: Workspace (Strategy, Initiatives, Projects, Circles)
│   │       └── Collapsible: Your Circles
│   │           └── Collapsible (per circle)
│   │               ├── Sessions > Previous / Upcoming
│   │               ├── Decisions
│   │               └── Commitments
│   └── SidebarInset
│       ├── [sticky] PageHeader (injected by current page via usePageHeader)
│       └── ScrollArea
│           └── <Outlet /> → current page
```

## Inbox `/`

Shows proposed decisions from the user's circles, filterable by circle.

```
PageHeader (via usePageHeader)

│   ├── PageHeaderGroup: "Inbox" + circle filter tabs + [+] button

└── PageHeaderRow
    ├── PageHeaderGroup: Filter popover
    └── PageHeaderGroup: Display popover

Page body
└── DecisionsTable (noDividers)
```

## Decision Detail `/circles/:slug/decisions/:decisionSlug`

Full decision view with tabs (overview, commitments, activity) and right sidebar.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    ├── PageHeaderGroup
    │   ├── Circle icon
    │   ├── Breadcrumb: "Circle > Decisions > DEC-N" + statement
    │   └── Tab buttons: Overview | Commitments | Activity
    └── PageHeaderGroup
        └── Sidebar toggle (PanelRight icon)

Page body (conditional on active tab)
├── Overview tab (max-w-800)
│   ├── PageTitle (editable, decision statement)
│   ├── Properties row
│   │   ├── PropertySelect (status, with Badge icons)
│   │   ├── Age indicator (colored)
│   │   ├── PropertySelect (owner, with Avatar icons)
│   │   ├── "Requested by" label
│   │   └── Circle label
│   ├── "Rationale" → MarkdownEditor
│   ├── "Implications" → bullet list
│   └── "Discussion" → MarkdownEditor
├── Commitments tab
│   └── CommitmentsTable
└── Activity tab (max-w-800)
    └── ChangeLog

DetailSidebar (fixed right, 288px, toggle)
├── SidebarSection "Properties"
│   ├── PropertyRow "Status" → Badge
│   ├── PropertyRow "Age" → colored text
│   ├── PropertyRow "Owner" → PropertySelect
│   ├── PropertyRow "Requested" → text
│   └── PropertyRow "Circle" → Link
└── SidebarSection "Summary" → text
```

## Commitment Review `/circles/:slug/commitments/:commitmentSlug`

Commitment view with tabs (overview, activity) and right sidebar.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    ├── PageHeaderGroup
    │   ├── Circle icon
    │   ├── Breadcrumb: "Circle > Commitments > COM-N" + outcome
    │   └── Tab buttons: Overview | Activity
    └── PageHeaderGroup
        ├── [Re-commit?] button (if rolled/at-risk)
        └── Sidebar toggle

Page body (conditional on active tab)
├── Overview tab (max-w-800)
│   ├── Alert banner (rolled = red, at-risk = amber)
│   ├── PageTitle (editable, commitment outcome)
│   ├── Properties row: Badge (status), due date, owner, circle
│   └── "Linked Decision" → Link card with Badge + status
└── Activity tab (max-w-800)
    └── ChangeLog

DetailSidebar (fixed right, 288px, toggle)
├── SidebarSection "Properties"
│   ├── PropertyRow "Status" → Badge
│   ├── PropertyRow "Due" → text
│   ├── PropertyRow "Owner" → text
│   └── PropertyRow "Circle" → Link
└── SidebarSection "Linked Decision" → Link + Badge
```

## Session Detail `/circles/:slug/sessions/:sessionNumber`

Meeting workspace with decisions agenda, commitments review, notes editor, and decision drawer.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    └── PageHeaderGroup: circle icon + "Circle > Session N" + friendly date

Page body (max-w-800)
├── Input (editable session title, styled as h1)
├── "Decisions" section
│   ├── Section header + [Add Decision] button
│   └── DecisionsTable
│       (showCircle=false, showAge=false, showSelect=false,
│        showDragHandle=false, showHeader=false, combinedStatusOwner,
│        noDividers, onReorder, onDecisionClick → opens drawer)
├── "Commitments" section
│   ├── Section header + [Add Commitment] button
│   └── CommitmentsTable
│       (showCircle=false, showDecision=false, showSelect=false,
│        showHeader=false, combinedStatusOwner, noDividers)
└── "Meeting Notes" section
    └── MarkdownEditor (session notes)

Decision drawer (fixed right, 480px, toggle on row click)
├── Drawer header: ref + statement + [open full] Link + [close] button
└── Drawer content
    ├── Properties row: Badge (status), age, owner, requested by
    ├── "Rationale" → text
    ├── "Implications" → bullet list
    └── "Discussion" → text
```

## Project Detail `/projects/:projectId`

Project detail with tabs (overview, decisions, commitments) and right sidebar.

```
PageHeader (via usePageHeader)
└── PageHeaderRow
    ├── PageHeaderGroup
    │   ├── Breadcrumb: "Projects >" + title
    │   └── Tab buttons: Overview | Decisions | Commitments
    └── PageHeaderGroup
        ├── [Follow/Following] button (public projects, non-owner)
        └── Sidebar toggle

Page body (conditional on active tab)
├── Overview tab (max-w-800)
│   ├── Alert banner (if >2 rolled commitments)
│   ├── PageTitle (editable, project title)
│   ├── InlineText (description, as p)
│   ├── Properties row
│   │   ├── PropertySelect (health, with colored icons)
│   │   ├── PropertySelect (visibility: public/private)
│   │   ├── PropertySelect (owner, with Avatar icons)
│   │   └── Link to parent initiative
│   └── "Teams" → TeamChips
├── Decisions tab
│   └── DecisionList
└── Commitments tab
    └── CommitmentList

DetailSidebar (fixed right, 288px, toggle)
├── SidebarSection "Properties"
│   ├── PropertyRow "Health" → HealthIndicator
│   ├── PropertyRow "Visibility" → PropertySelect
│   ├── PropertyRow "Owner" → PropertySelect
│   └── PropertyRow "Initiative" → Link
├── SidebarSection "Signals"
│   ├── PropertyRow "Open" → commitment count
│   ├── PropertyRow "Rolled" → rolled count
│   └── PropertyRow "Dec. (14d)" → recent decision count
├── SidebarSection "Teams" → TeamChips
└── SidebarSection "People" (private projects only)
    ├── Owner row: Avatar + name + "Owner" Badge
    ├── Subscriber rows: Avatar + name + [remove] button
    └── Select (invite person)
```
